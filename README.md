# My Image Gallery ✨

A responsive, dynamic, and aesthetic image gallery web application built using **HTML5**, **CSS3**, and **JavaScript**. This project features category filtering,used javascript for next/prev button and interactive lightbox popup modal,added hover effects and smooth transitions,ensures responsive design for different screen size and custom end-of-album notifications.



## 🌟 Features

- **🏷️ Album Category Filtering:**
  
   Easily filter photos by categories including:
  - 🌿 Nature Magic
  - 🐾 Cute Baby Pets
  - 🎉 Fun & Outings
  - 👗 Girls Fashion

 - **✨ Hover Effects & Smooth Transitions:**
   
   Micro-interactions featuring glowing card borders, subtle image scaling, and fluid transitions on hover.
   
- **🔍 Interactive Lightbox View:**
  
  Click on any image to open a high-resolution, full-screen popup view.
  
- **⬅️➡️ Seamless Navigation:**
  
  Navigate through images using Previous (`‹`) and Next (`›`) buttons within the lightbox.
  
- **📱 Responsive Grid Layout:**
  
  Built using CSS Grid for an optimal viewing experience across mobile, tablet, and desktop screens.
  
- **🚨 End-of-Album Modal Alert:**
  
   Custom popup card alerting users when they reach the end of an album with options to restart or exit.

---

## 🛠️ Technologies Used

- **HTML5:**
  Semantic structure and content organization.
  
- **CSS3:**
  Flexbox, CSS Grid, Glassmorphism backdrop filters, custom animations, and glowing keyframes.
  
- **JavaScript:**
  Event listeners, DOM manipulation, state management for lightbox and category filtering logic.
